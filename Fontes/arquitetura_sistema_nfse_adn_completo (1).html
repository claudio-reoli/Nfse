3. Endpoints essenciais da API REST (ADN / Sefin)
Emissão POST /nfse — Gera NFS-e a partir da DPS · síncrono · retorno imediato com autorização ou rejeição
Consulta NFS-e GET /nfse/{chaveAcesso} — Retorna XML completo comprimido (GZip) em Base64
Consulta DPS HEAD /dps/{id} — Verifica se DPS já foi autorizada · base da estratégia de idempotência
Eventos (escrita) POST /nfse/{chave}/eventos — Cancelamento · substituição · manifestação · vinculação de pagamento
Eventos (leitura) GET /nfse/{chave}/eventos — Lista histórico de eventos por tipo e número de sequência
DANFSe GET /danfse/{chaveAcesso} — PDF auxiliar formatado para compartilhamento com o tomador
Distribuição DF-e GET /DFe/{NSU} — Lotes de até 50 documentos · NSU sequencial · polling do tomador
Parâmetros municipais GET /parametros_municipais/{codigoMunicipio} — Alíquotas · regimes especiais · retenções · benefícios fiscais

6. Resiliência, idempotência e observabilidade
Idempotência Chave: idDps + CNPJ + série + número — previne dupla autorização em cenários de retry
Retry Backoff exponencial com jitter aleatório · circuit breaker após 3 falhas consecutivas · fila de reprocessamento assíncrona
Webhooks Recepção proativa de DF-e e eventos pelo tomador — substitui polling contínuo dos endpoints
Cache Parâmetros municipais com TTL configurável (4–24h) + invalidação event-driven — alíquotas raramente mudam
Observabilidade Logs estruturados JSON (traceId · spanId · cnpj · evento · latência) · OpenTelemetry · alertas P1/P2 automáticos
TLS / mTLS TLS 1.2+ obrigatório · mTLS para integrações B2G · certificado ICP-Brasil e-CNPJ obrigatório para DPS
7. Numeração, série e regras de controle
Série Exclusivamente numérica a partir de 01/01/2026 — série alfanumérica aceita apenas para competências ≤ dez/2025
Pulos de NSU Decorrem de falhas técnicas ou concorrência no ADN — não geram inconsistência contábil e não devem causar rejeição no sistema receptor
Arredondamento Half-even (bancário) obrigatório para PIS/COFINS desde NT 007/2026 — tolerância de R$ 0,01 no motor de cálculo
Imutabilidade NFS-e autorizada não pode ser alterada — apenas cancelada (irreversível) ou substituída por nova DPS
Prefixos de erro Prefixo E — erros originados no ADN nacional  ·  Prefixo X — erros originados no sistema municipal (vigência 01/01/2026)
